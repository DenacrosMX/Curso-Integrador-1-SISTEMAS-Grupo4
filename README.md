# Curso-Integrador-1-SISTEMAS-Grupo4

Proyecto de Gestión Digital para Condominios y Edificios Residenciales.

## Descripción
Sistema web desarrollado en React para la gestión de pagos, incidencias y comunicados en condominios.

## Tecnologías
- React (Vite)
- JavaScript
- Tailwind CSS

## Módulos
- Login
- Dashboard
- Pagos
- Incidencias
- Comunicados
